I: Closet Search Filter Web/Mobile App

  • Whether you're a travel lover like me and find yourself constantly living out of a suitcase or simply just need a better way to go through your clothes stuffed drawers - I decided to use JavaScript React to create a clothing search filter. This app allows me to be able to scroll through my items as well as filter the results depending on type or color. I included functions such as "pin" and "unpin" in order to see different clothing items next to each other. There is also a hide button to prevent the item from appearing wheather it's because it's a no or because it's in the dirty bin. Skills Performed: HTML/CSS, JavaScript React, filtering arrays, Class Based Components

II: Personal Online Blog

  • Being passionate about the practice of journaling I decided to create a simple online blog platform using JavaScript React and Google Firebase. I designed the homepage to include a photo of a journal overlayed with some writing I did on SketchPad to evoke that handwritten journal feel. Under "thoughts" I have the entries being transferred from Firebase appearing in order from newest to oldest. Lastly, you can create a new entry under the "new" tab that will add onto the Firebase data collection at publish. I kept the design clean and simple embodying that of a blank notebook. Skills Performed: HTML/CSS, JavaScript React, React Hooks: useState and useEffect, Google FireBase
